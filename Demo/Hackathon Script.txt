## Hackathon Demonstration Script

## Intro Slide Video


Institutions are ready for DeFi but remain paralyzed by the risk of non-compliant liquidity, which we solve with the Agentic Compliance Bridge—a privacy-preserving firewall that only allows cryptographically verified actors into the pool. It operates as an automated gatekeeper by locking funds from non-compliant users and releasing them only after Zero Knowledge Proofs confirm they meet strict wealth and residency requirements, verified via Chainlink’s off-chain environment and on-chain attestations.

On the far right, we have the Bridge API orchestrating ZKPs and CCIP across three terminals.

Terminal 1 simulates the Trad-Fi bank holding raw PII (200 ETH, USA).

Terminal 3 is our operator interface where we execute the simulation to bridge the data and release the vault.

Phase 1:

10 ETH is sent from the user to the Agentic Compliance Bridge. It is not yet flagged as compliant so the funds are locked into an escrow holding vault.

COMMAND 2- Bottom Terminal 3
It shows our recipient wallet address inside the compliant DeFi ecosystem as well as the empty wallet balance of 0.01 ETH

Phase 2:
COMMAND 3
This command instructs the Agentic Compliance Bridge to audit the bank's credentials, originate and verify a Zero Knowledge Proof, and broadcast it via CCIP to enable instant atomic settlement of assets already secured on the destination chain.

We used the Circom 2.0 compiler to define our private logic, requiring the user to have over 100 ETH and US residency, while ensuring the bank never leaks sensitive PII. 

The Agentic Bridge API in Terminal 2 utilizes SnarkJS as a proving engine to generate a ZKP off-chain, so raw data never touches the public ledger. The proof is sent across chain using CCIP to our Solidity verifier contract that confirms: yes or no that the user is credentialed.
